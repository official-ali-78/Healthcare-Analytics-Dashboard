# Healthcare Analytics Dashboard

## Project Overview

The Healthcare Analytics Dashboard is designed to analyze healthcare data and provide insights into patient demographics, medical conditions, hospital performance, billing trends, and insurance provider contributions.

The project combines SQL for data analysis and Power BI for interactive data visualization, enabling stakeholders to monitor healthcare operations and identify important trends.

## Objectives

* Analyze patient demographics and medical conditions.
* Evaluate hospital and doctor performance.
* Monitor billing and revenue trends.
* Assess insurance provider contributions.
* Measure patient length of stay.
* Build an interactive dashboard for decision-making.

## Tools & Technologies

* MySQL
* Power BI
* SQL
* DAX

## Key Insights

* Identified the most common medical conditions among patients.
* Analyzed revenue generated by hospitals and doctors.
* Evaluated average patient billing and stay duration.
* Compared insurance provider contributions.
* Built KPIs and visualizations to support healthcare decision-making.

## Dashboard Features

* Executive Overview
* Hospital Performance Analysis
* Medical Condition Analysis
* Insurance Analysis
* Length of Stay Analysis
* Interactive Filters and Drill-Down Capabilities
